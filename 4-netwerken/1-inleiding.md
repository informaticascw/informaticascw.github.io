# Inleiding

## Leerdoelen
; de onderwerpen zijn redelijk compleet
; de werkwoorden die aangeven tot op welk niveau (RTTI, Bloom etc) leerlingen ermee moeten kunnen werken ontbreken nog

Lagenmodel
- OSI model (of TCP/IP model)
- Naam en functionaliteit per laag
- Concept van inpakken/uitpakken van data uit de onderliggende/bovenliggende laag

Protocollen & begrippen per laag
- Protocollen: HTTP, DNS + DHCP, SMTP + POP + IMAP

- Protocollen: TCP + UDP, IP, RIP
- Begrippen: IP-adres, subnetmask, broadcast-adres, netwerk-adres, default gatewau

- Protocollen: ARP
- Begrippen: MAC-adres, broadcast-adres

- Begrippen: Bandbreedte, pre-amble, CRC

Per protocol:
- In welke laag past dit protocol?
- Welke functie heeft dit protocol?
- Hoe werkt het protocol op hoofdlijnen?

- Welke apparaten (modem, repeater, switch, acces-point, router, firewall) doet wat en op welke laag?\
(onderscheid tussen functionaliteit en "doosje" maken, sommige "doosjes" hebben de naam van 1 functie maar herbergen meerdere functies, denk bijvoorbeeld aan je "modem" thuis, wat ook een router is)

- Media op de fysieke laag (twisted pair, wifi, ...)
